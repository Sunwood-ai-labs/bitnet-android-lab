# Limitations

- This repository documents a single-device lab result, not a broad Android compatibility claim.
- The successful path depended on local Termux-specific workaround patches.
- The fast TQ2 inference result came from an intermediate checkpoint artifact.
- Evidence shows one short non-empty completion, not a quality benchmark.
- Any published `tok/s` figures in this repo are single-run throughput references, not benchmark medians or sustained-speed claims.
- Shutdown emitted a `FORTIFY` warning.
- Some original raw lab notes contained private paths and device identifiers; the public repo removes or sanitizes those.
