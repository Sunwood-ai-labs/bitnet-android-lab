# Evidence Folder

This folder stores sanitized evidence snippets captured during the March 23, 2026 lab session and March 25, 2026 follow-up reruns.

- [`manifest.md`](./manifest.md) maps public claims to scripts and evidence files.
- [`logs/`](./logs/) contains curated text snippets from the Termux-side runs.

These files are not full raw logs. They are trimmed to the minimum lines needed to support the public claims made in the README and docs, including the phase-separated `tok/s` references.

Machine-specific paths were normalized where needed so the evidence can be published without exposing local Termux paths.
